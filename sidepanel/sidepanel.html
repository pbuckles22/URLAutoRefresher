URL Auto Refresher

Use the toolbar icon anytime to return here. The side panel shows the same lists for quick monitoring. Configure global groups (multi-tab) and individual jobs below.

Display

Global (0)

Saved groups are listed first. To add a group, browse windows and tabs below, check which tabs belong to it, set each tab’s refresh URL (defaults to that tab’s current address when you build the list), then Save. One shared schedule refreshes every included tab together. On each tick the extension loads the saved URL for that tab (e.g. after a raid, it returns to the channel you configured—not the tab’s live URL unless you saved that).
