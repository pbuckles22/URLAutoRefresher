URL Auto Refresher

Use the toolbar icon anytime to return here. The side panel shows the same lists for quick monitoring. Configure global groups (multi-tab) and individual jobs below.

Display

Global (0)

Browse windows and tabs, check which tabs belong to the group, set each tab’s refresh URL, then Save. One shared schedule refreshes every included tab together (Epic 4.1).

Uses all windows (windows.getAll).

      Individual (0)

      Pick a tab, set the URL to load on each refresh, interval and jitter (seconds), then Save.
